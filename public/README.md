# 🌿 Carbon Footprint Dashboard — KBTU

Веб-дашборд для визуализации углеродного следа Казахстанско-Британского Технического Университета.

## 📊 Функционал

- Обзор ключевых показателей CO₂e
- Потребление электроэнергии по зданиям и месяцам
- Выбросы Scope 1 и Scope 2
- Сравнительный анализ зданий КБТУ
- Прогноз снижения выбросов до 2030 года
- Поддержка трёх языков: RU / KZ / EN

## 🏢 Здания КБТУ

- Толе би, Панфилова, Абылай хана
- Библиотека, Цокольный этаж, 5 этаж, Столовая

## 🛠️ Стек

- React 18 + TypeScript
- Chart.js
- Vite
- Inter Font

## 🚀 Запуск

npm install
npm run dev

## 📐 Методология

Коэффициенты выбросов: Министерство экологии РК, IPCC AR6
- Scope 1: 0.202 кг CO₂e/кВтч (тепло/газ)
- Scope 2: 0.514 кг CO₂e/кВтч (электроэнергия, Казахстан 2023)
